INTRUCCIONES DE USO
1. Iniciar la consola de Windows
2. Ubicarnos en la carpeta donde se encuentra el programa
3. Escribir en la consola "javac conversionMorse.java" sin las comillas, dar enter
4. Seguidamente Escribir "java conversionMorse" sin las comillas, dar enter
5. Escoja entre las opciones que le aparecen 1 0 2
6. Si escogió 1 entonces escriba la palabra para que se convierta a morse
7. Para volver a iniciar el programa repita los pasos 3 y 4
8. Si escogió 2 entonces escriba la clave morse para que se convierta en palabras
9. FIN
